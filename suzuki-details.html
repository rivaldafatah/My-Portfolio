QA Testing for Printing Service Ordering

Overview: In this project, I conducted thorough testing of the Printing Service Ordering Application. The focus was on ensuring the functionality, performance, and security of the application.

Testing Types

  • Manual Testing for feature functionality
  • Regression Testing for existing features after updates
  • Performance Testing under high user load
  • Security Testing to ensure data protection

Results

This project helped identify and resolve over 30 critical bugs before release, ensuring a smooth user experience and delivering a high-quality product to the users.

Multiple issues were detected and resolved in key areas, ensuring optimal performance and security.
